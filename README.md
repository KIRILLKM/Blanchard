# Верстка семантичного, адаптивного и интерактивного сайта для художественной галереи
---

## Описание проекта

«Blanchard» — современный одностраничный сайт для вымышленной художественной галереи.

**Главные особенности:**

- Адаптивность: от 320px до 1920px
- Семантика и доступность: правильные HTML5-теги, ARIA-атрибуты, управление с клавиатуры
- Минимальный JS: логика на чистом JavaScript без фреймворков
- Бургер-меню с блокировкой скролла
- Кастомный селект (Choices.js)
- Слайдеры (Swiper.js)
- Модальные окна с кастомным скроллбаром (SimpleBar)
- Тултипы (Tippy.js)
- Яндекс.Карты с кастомной меткой
- Форма обратной связи с валидацией

---

## Стек технологий

**Верстка:** HTML5, CSS3 (Flexbox, Grid), CSS Custom Properties, БЭМ

**JavaScript:** Vanilla JS, ES6+

**Библиотеки:** Swiper, Choices.js, SimpleBar, Tippy.js, Inputmask, JustValidate, Yandex Maps API

**Инструменты:** PHP, PHPMailer

---

## Ключевые особенности

### Адаптивность и семантика

- Адаптивные изображения через `<picture>` и `srcset`
- 6 брейкпоинтов: 320px, 576px, 768px, 992px, 1200px, 1920px
- Семантические теги: header, main, section, article, nav, address, footer
- ARIA-атрибуты для скринридеров

### Интерактивные компоненты

- Кастомный селект Choices.js для фильтрации галереи
- Слайдер Swiper.js с пагинацией в виде дробей
- Аккордеон с табами для смены информации о художниках
- Модальные окна с кастомным скроллбаром SimpleBar

### Карта и формы

- Яндекс.Карты с кастомной меткой, отключены лишние элементы управления
- Форма обратной связи: валидация JustValidate, маска телефона Inputmask
- Отправка данных на email через PHP + PHPMailer

## Запуск проекта

**1. Клонируйте репозиторий**
```
https://github.com/KIRILLKM/Blanchard.git
```
**2. Настройка формы**
В файле `mail.php` необходимо указать свои SMTP-данные:
```
$mail->Host       = 'smtp.gmail.com';        // SMTP-сервер
$mail->Username   = 'your-email@gmail.com';  // Логин
$mail->Password   = 'your-app-password';     // Пароль или app-пароль
$mail->addAddress('your-email@gmail.com');   // Куда приходят письма
```
**3. Запустите локальный сервер (например, OpenServer или MAMP) для работы PHP-обработчика форм**
**4. 1. Откройте `index.html` в браузере**
